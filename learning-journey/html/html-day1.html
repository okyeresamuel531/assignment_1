What is HTML?

HTML stands for Hypertext Markup Language. It is the language used to structure and display content on the web. Every webpage you visit is made with HTML. The structure of an HTML document is formed by elements, which consist of tags that tell the browser how to display content. This is what we call a "markup" language.

In this lesson, we will go through the basic structure of an HTML document and understand how tags work to form that structure.

Day 1: HTML Structure & Tags

Letâ€™s explore the structure of an HTML document. Every HTML document starts with a doctype declaration, followed by the <html> tag. Inside the <html> tag, there are two main sections: the <head> and the <body>.

Basic HTML Structure

<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>My First HTML Page</title>
    </head>
    <body>
        <h1>Welcome to My First Web Page</h1>
        <p>This is my first webpage using HTML!</p>
    </body>
</html>

Welcome to My First Web Page

This is my first webpage using HTML!

In the example above:

  • <!DOCTYPE html>: Defines the document type and version of HTML (HTML5 in this case).
  • <html>: The root element that contains all other HTML elements.
  • <head>: Contains meta-information about the document like character set, viewport settings, and the title of the page.
  • <body>: Contains the actual content of the webpage, such as headings and paragraphs.
  • <h1>: Defines a top-level heading.
  • <p>: Defines a paragraph of text.
